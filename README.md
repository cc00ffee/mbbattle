# Complimentary Versus Battle

Экспериментальный комплиментарный versus-баттл в стихах.  
Все тексты — авторские, с иронией, каламбурами и отсылками.

## Формат
- Каждый раунд хранится в отдельном `.md` файле в папке `rounds/`.
- Список раундов и краткие пояснения — в `meta/tracklist.md`.

## Лицензия
Тексты распространяются по лицензии [Creative Commons BY-NC-SA 4.0](LICENSE).  
Это значит:
- можно читать, делиться и делать ремиксы,
- только с указанием автора,
- нельзя использовать в коммерческих целях,
- производные работы должны распространяться на тех же условиях.
